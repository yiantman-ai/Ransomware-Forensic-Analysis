# Quick Start Guide

## Start Server
```bash
cd web_infrastructure
python3 c2_server.py
```

## Test Endpoints
```bash
# Phishing page
curl http://localhost:8080/

# Loader script
curl http://localhost:8080/loader.ps1

# Config file
curl http://localhost:8080/config.dat
```

## Security Reminder

ONLY RUN IN ISOLATED LAB ENVIRONMENT
